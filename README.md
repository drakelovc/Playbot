# Playerok Steam Guard Bot

Бот для автоматизации продаж на Playerok с интеграцией Steam Guard кодов.

## Возможности

- **Управление аккаунтами** — добавление через токен/email, мультиаккаунтинг
- **Автопроцессы** — автовыдача кодов, автоподтверждение, авто-восстановление лотов, авто-поднятие
- **Ручные операции** — вывод баланса, чтение чатов, рассылка
- **Профиль аккаунта** — товары, продажи, отзывы, чаты
- **Статистика** — экспорт в XLSX/TXT, фильтрация по периодам
- **Настройки** — прокси, куки, чёрный список
- **Модули (плагины)** — расширяемая система модулей с включением/выключением
- **Мониторинг чатов** Playerok в реальном времени
- **Вечный онлайн** — поддержание статуса онлайн на Playerok
- **Ежедневная сводка** — автоматическая отправка статистики

## Структура проекта

```
main.py                  # Точка входа
core/
  bot_instance.py        # Экземпляр Telegram бота
  config.py              # Конфигурация и дефолты
  steam_guard.py         # Steam Guard: загрузка maFile, генерация кодов
  playerok_connection.py # Подключение к Playerok API
  event_loop.py          # Обработка событий Playerok
  background.py          # Фоновые задачи
  helpers.py             # Утилиты (сводка, аналитика)
  logging_setup.py       # Настройка логирования (plain + JSON)
handlers/
  main_menu.py           # /start, нижняя клавиатура
  profile.py             # Личный кабинет, финансы, подписка
  accounts.py            # Мои аккаунты: список, добавление
  auto_processes.py      # Автопроцессы: товары, чаты, финансы
  manual_ops.py          # Ручные операции
  account_profile.py     # Профиль: контент, покупки, товары, продажи, отзывы, чаты
  statistics.py          # Статистика с экспортом XLSX/TXT
  settings.py            # Прокси, куки, чёрный список, токен
  modules.py             # Управление модулями/плагинами
  sales.py               # Мои продажи (по статусам)
  items.py               # Мои товары (активные/завершённые)
  chats.py               # Мои чаты
  actions.py             # Ответы, отправка кодов, загрузка .maFile
plugins/                 # Система плагинов
playerokapi/             # Playerok API библиотека
db.py                    # Postgres-бэкап для Railway
```

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте `.env` файл на основе `.env.example`:
```
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_telegram_id
```

2. (Опционально) Для Railway добавьте `DATABASE_URL`.

## Запуск

```bash
python main.py
```

Или через Procfile (Railway/Heroku):
```
worker: python main.py
```

## Steam Guard

Загрузите `.maFile` файлы в папку `accounts/` или отправьте их боту через Telegram.
Бот автоматически распарсит `shared_secret` и будет генерировать коды по команде `!code`.
